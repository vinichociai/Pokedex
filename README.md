# Pokedex

Projeto web desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de praticar desenvolvimento front-end, manipulação do DOM e exibição dinâmica de informações relacionadas a Pokémon.

## Sobre o projeto

Este projeto foi desenvolvido como estudo prático de front-end, utilizando uma interface simples para exibir informações de Pokémon de forma visual e organizada.

A aplicação trabalha com arquivos estáticos e pode ser executada diretamente no navegador.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Funcionalidades

* Interface visual de Pokédex
* Exibição de informações de Pokémon
* Manipulação de elementos com JavaScript
* Organização de arquivos estáticos
* Publicação via GitHub Pages

## Como executar

Clone o repositório:


git clone https://github.com/vinichociai/Pokedex.git


Acesse a pasta do projeto:


cd Pokedex


Abra o arquivo `index.html` no navegador.
